# Android-Games
Current progress of android games in development (Not including GameMaker projects)

Nappy Bird
	Somewhat similar to ‘Flappy Bird’. Released and on Google Play store for free. Tap to jump, avoid ‘MyQuil’ tablets, pillows, and milk while collecting anvils and coffee. Don’t let green meter run out.

Name Pending
	Tower building game in progress. Not yet published on Google Play store. Place and rotate different shapes to build a tower as tall and structurally stable as possible. Wind comes after set amount of moves to knock tower down. Score is based on height after wind.
